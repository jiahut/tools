###  简单说明

1. command line dict
  * 一只百度爬虫
  * 命令行下的字典查询工具
  * 目前ruby和python两个语言版本的实现
2. sublime project manager
  * 管理(CRUD)sublime的project上下文配置
3. svn lines statistics
  * 统计svn的代码变更
4. capturer
  * 通过规则url自动下载网络资源
5. log_handler
  * log handler by replace with regex
6. 动态db helper
  * 动态生成单表数据库表名
7. api-client 
  * 模拟客户端发送请求
